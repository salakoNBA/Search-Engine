Time format

You are given a number that represents the number of seconds passed since 1.1.1970.
Write a program that calculates the current time, i.e., find the hours, minutes and seconds of the given number of seconds.

Format: hours:minutes:seconds

Example: 14:9:7

You don't need to import anything, just use % and / and don't forget about days!



Memory limit: 256 MB
Time limit: 8 seconds
